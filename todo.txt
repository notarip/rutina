---------------------------------------------
entidades:

usuario: usuarios genericos del sistema
rol: rol que puede tener un usuario
campo: campoos de entrenamiento 
entrenamiento: entrenamiento, agrupa varios done, puede relacionarce con una rutina o se libre
rutina: rutina que tiene un usuario esta compuesta de varios ejercicios
ejercicio: todos los ejercicios que se pueden planificar en una rutina. 
evetos: eventos genericos, en principio son comentarios que se pueden asiociar a cualquier entidad

sintesis:
Es un sitio muy orientado a movile que permite a los usuarios y entrenadores gestionar los entrenamientos.


----------------------------------------------

historias:


- como usuario: darme de alta en el sistema, con nombre, email y password
- como usuario: asociar mi cuenta a un campoo 
- como campo: recibir la informacion de todos los usuarios que me tienen asociados
- como campo: poder cargar las rutinas de mis usuarios
- como usuario: poder cargar mis entrenamientos
- como usuario: poder elegir si entreno libre o voy a hacer una rutina
- como usuario: quiero recibir una notificacion si mi coach cambia algo de mi rutina !!
- como campo: poder cambiar la rutina y comentar los entrenamientos !!
- como usuario: incluir el los comments a otro usuario(tipo twitter)

Escenario: Un entrenador carga una rutina para un usuario
Dado que: 
Un usuario tiene asociado un campo
Y que el campo tiene asociado un entrenador
Y que el usuario tiene el certificado medico al dia
Y que el usuario tiene la cuota al dia
Cuando: 
Un entrenador cargue una rutina para un usuario
Y complete todos las Sesiones de ejercios
Entonces:
Se debera validar que:
- Los ejercicios cargados no trabajan los grupos musculares restringidos en el usuario, si es que tiene. Advertencias
- Si las sesiones de ejercicios superan el tiempo que puede dedicar el usuario a cada entrenamiento advierta.
- Si tiene más de 60 años se enviara una notificación al administrador del campo notificando la rutina creada.

***
Escenario: Un usuario carga un Entrenamiento asociado a una rutina
Dado que:
Un usuario tiene asociado un campo
Y tiene asociada una rutina
Y tiene el certificado medico al dia
Y que el usuario tiene la cuota al dia
Cuando:
El usuario carga un entrenamiento para una rutina
Y especifica las acciones que hizo
Entonces:
Se debera validar que: Los ejercicios cargados no trabajan los grupos musculares restringidos en el usuario, se permite el registro pero 
con advertencias.
Y se enviara una notificación al entrenador de los ejercicios hechos.
Y si tiene más de 60 años se enviara una notificación al administrador del campo notificando los ejercios hechos.

***

Escenario: Un usuario se genera su propia rutina
Dado que:
El usuario esta habilitado para generar sus propias rutinas
Cuando: 
El usuario se carga su rutina con sus sesiones
Entonces:
Se devera validar:
- Los ejercicios cargados no trabajan los grupos musculares restringidos en el usuario, si es que tiene. Se advierte pero se permite la carga.
- Si las sesiones de ejercicios superan el tiempo que puede dedicar el usuario a cada entrenamiento advierta.
- Se advierte sobre el trabajo incorrecto de grupos musculares en las sesiones (ejemplo "Tienes más de 3 ejercicios de biceps para esta sesion")
- Se sugiere la carga de grupos musculares en las sesiones (ejemplo: "No estás trabajando biceps")
- Se advierte en las Series si los pesos superan más del 20 % de lo ultimo que tiene registrado para ese ejercicio
