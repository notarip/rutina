---------------------------------------------
entidades:

usuario: usuarios genericos del sistema
rol: rol que puede tener un usuario
gimnasio: gimnasioos de entrenamiento 
entrenamiento: entrenamiento, agrupa varios done, puede relacionarce con una rutina o se libre
rutina: rutina que tiene un usuario esta compuesta de varios ejercicios
ejercicio: todos los ejercicios que se pueden planificar en una rutina. 
evetos: eventos genericos, en principio son comentarios que se pueden asiociar a cualquier entidad

----------------------------------------------
Sintesis:

Es un sitio en que los usuarios pueden asociarce a gimnasios para llevar un resgistro de las rutinas de ejercicios que
realizan y que los entrenadores puedan hacer un seguimiento personalizado presencial y no presencial de los entrenamientos
de los usuarios del gimnasio.


***
Escenario: Un entrenador carga una rutina para un usuario
Dado que: 
el usuario tiene el certificado medico al dia, si no lo tiene, se advierte(!)
Y que el usuario tiene la cuota al dia, si no la tiene, se advierte(!)
Cuando: 
Un entrenador cargue una rutina para un usuario
Y complete todos las Sesiones de ejercios
Entonces:
- Si el usuario posee otra rutina para ese gimnasio se dara de baja
- Si los ejercicios cargados trabajan los grupos musculares restringidos en el usuario, se advierte(!)
- Si las sesiones de ejercicios superan el tiempo que puede dedicar el usuario a cada entrenamiento, se advierte(!).
- Si tiene más de 60 años se enviara una notificación al administrador del gimnasio notificando la rutina creada.

***
Escenario: Un usuario carga un Entrenamiento asociado a una rutina
Dado que:
Un usuario tiene asociado un gimnasio
Y tiene asociada una rutina
Y tiene el certificado medico al dia, si no lo tiene, se advierte(!)
Y que el usuario tiene la cuota al dia, si no la tiene, se advierte(!)
Cuando:
El usuario carga un entrenamiento para una rutina
Y especifica las acciones que hizo
Entonces:
- Si los ejercicios cargados trabajan los grupos musculares restringidos en el usuario, se advierte(!)
- Se informan los ejercicios que quedan pendientes para finalizar la rutina.
- Se aconseja con que ejercicio continuar
- Se enviara una notificación al entrenador de los ejercicios hechos.
- Si tiene más de 60 años se enviara una notificación al administrador del gimnasio notificando los ejercios hechos.

***

Escenario: Un usuario se genera su propia rutina
Dado que:
El usuario esta asociado o no a un gimnasio
Cuando: 
El usuario se carga su rutina con sus sesiones
Entonces:
Se devera validar:
- Si los ejercicios cargados trabajan los grupos musculares restringidos en el usuario, se advierte(!)
- Si las sesiones de ejercicios superan el tiempo que puede dedicar el usuario a cada entrenamiento, se advierte(!)
- Se advierte(!) sobre el trabajo incorrecto de grupos musculares en las sesiones (ejemplo "Tienes más de 3 ejercicios de biceps para esta sesion")
- Se sugiere la carga de grupos musculares en las sesiones (ejemplo: "No estás trabajando biceps")
- Se advierte en las Series si los pesos superan más del 50 % de lo ultimo que tiene registrado para ese ejercicio

***
Escenario: Carga de una rutina con inicio/fin
Dado que:
Un usuario quiere realizar un entrenamiento programado
Y se cargaron todas las sesiones con su correspondiente fecha
Y la rutina tiene una fecha de inicio y una de fin 
Cuando:
A medida que el usuario vaya cargando los entrenamientos
Entonces: 
Si la fecha del entrenamiento no coincide con la planificada, da la opcion de 
replanificar, lo que corre todas las rutinas X cantidad de dias.


***
Escenario: Un usuario carga una serie para un ejercicio
Dado que:
El usuario puede cargar una serie sea para una rutina o para un entrenamiento
Cuando: 
El usuario carga las series relacionadas a un ejercicio.
Entonces:
Se valida que los cuantificadores seleccionados estan habilitados para el ejercicio
